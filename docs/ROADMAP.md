# 로드맵 · 보완점

## 완료 (v0.2)
- ✅ **저장형 모집**: 자리는 연결과 무관하게 저장 — 창을 닫거나 나중에 들어와도 유지.
- ✅ **소유권 안정화**: 브라우저 고정 `clientId`(localStorage)로 소유 → 새로고침·재접속해도 본인 자리 유지·해제 가능.
- ✅ **설정 편집**: 시간(예정 시각)·티어(아이언~챌린저·상관없음)·큐(일반/칼바람/솔로/자유/5인/기타) 실시간 편집.
- ✅ **게임 수명**: 예정 시각까지 수정 가능, 지나면 자동으로 새 게임(빈 자리)으로 교체.
- ✅ 실시간 반영 유지 + 접속 중 자동 만료 반영(타이머).

## 지속성(중요)
현재 저장은 **인메모리**다. 서버가 도는 동안은 유지되지만 **재배포·재시작 시 초기화**된다.
- 잠들기 방지: 외부에서 `/healthz`를 주기적으로 호출(킵얼라이브) — [DEPLOY.md](DEPLOY.md) 참고.
- 완전 영속(재배포에도 유지): 저장 경계(`server/rooms.ts`)를 인터페이스로 분리해 두었으므로,
  Upstash Redis / Supabase 등 **외부 DB 어댑터**로 교체하면 된다.
